# Proposed Solution

Our proposed solution consists of two main components. First, a framework was developed to ensure that any AI developed by TD does not negatively impact employee well-being, privacy, and security. The framework is the backbone of our solution as it was designed to address the problem at hand. To ensure that our framework will be implemented properly, the second component created was a project management plug in. The plug in was created to be seamlessly integrated into popular software project management tools such as Jira. The plug in scans each ticket that is created in jira and tests it against our framework. Any incompliances found by the plug-in are flagged to the user with the specific breach of the framework highlighted. Together, both the framework and plug-in protect the well-being, privacy, and security of all TD employees as new AI technologies are developed.
